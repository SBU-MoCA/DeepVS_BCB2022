# DeepVS_BCB2022
## Repo structure

The repository structure is described below. 

```
├── README.md                                 : Description of this repository
├── models                                    : folder contains model design.
│   ├── __init__.py                           : init
│   ├── conv_attn_torch.py                    : model design of conv_attn 
├── checkpoints                               : config files and checkpoints of the pretrained models
│   ├── cnv2_attn_ts_rrhr_1y                  : code to scan for all Bluetooth devices that are connected
│   ├── cnv2_attn_ts_rrhr                     : code to print ou characteristics when connected to the oximeter
├── LICENSE

```
